# Claude Code

> 官方文档：https://code.claude.com/docs/en/overview

## 安装步骤

```md
# macOS / Linux（推荐）
curl -fsSL https://claude.ai/install.sh | bash

# Windows（推荐）
irm https://claude.ai/install.ps1 | iex

# Windows（WinGet）
winget install Anthropic.ClaudeCode

# macOS（Homebrew）
brew install --cask claude-code
```

## 使用方法

```md
cd /project

终端启动命令
[claude: 当前目录启动 Claude Code 交互会话]
[claude -c: 继续上一次会话]
[claude -p "指令": 非交互直接执行任务]
[claude -r 会话 ID: 恢复指定历史会话]

会话操作斜杠命令
[/help: 查看全部帮助命令]
[/clear: 清空当前对话上下文]
[/compact: 压缩会话历史、省 Token]
[/rewind: 回退到上一步代码状态]
[/exit: 退出 Claude Code 会话]

模型与配置
[/model: 切换 Sonnet/Opus/Haiku 模型]
[/config: 打开全局配置]
[/status: 查看会话、路径、模型状态]
[/cost: 查看当前会话 Token 消耗]
[/doctor: 环境 & 连接问题诊断]
[/effort: 设置思考深度 high/medium/low]

项目记忆 & 任务
[/init: 生成项目 CLAUDE.md 记忆文件]
[/memory: 编辑项目记忆 CLAUDE.md]
[/todos: 管理项目待办任务列表]

插件管理
[/plugin marketplace add <repo>: 添加插件市场]
[/plugin install <name>@<marketplace>: 安装插件]
[/plugin list: 查看已安装插件]
[/reload-plugins: 重新加载插件]

快捷工具
[/copy: 复制最近一次回答到剪贴板]
[/export 文件名: 导出整段对话存档]
[/add-dir: 关联额外项目目录]
[/permissions: 管理命令执行权限]
[/usage: 查看额度与限流状态]
```
